# NAME

close-duplicate-tabs

# SYNOPSIS

adds a toolbar button that shows the number of duplicate tabs and clicking it allows to quickly close them 
click the toolbar button or set and use the shortcut to close duplicate tabs 

# USER INPUTS

none

# OPTIONS

none

# REQUIRED PERMISSIONS

- **tabs**: used to access tab properties and hook into tab event
- **cookies**: used to read the tab containerId property (container <=> cookieStore) 

# OPTIONAL PERMISSIONS

none
