
# NAME

**gather-from-tabs** - run userscripts against multiple tabs on demand and gather their output  

# SYNOPSIS

Allows to create and run userscript that you can run on useraction/demand against one or multiple tabs.

# USER INPUTS

- userscript
- tabs

# OPTIONS

Granting the addon the optional notifications permissions allows it to show a notification when the timer ends.

# REQUIRED PERMISSIONS

- **clipboardWrite**: used to copy output into the clipboard
- **menus**: used to add the right click context menu
- **storage**: used to store the userscripts 
- **downloads**: used to init the download of links after selection
- **<all_urls>**: used to access multiple tabs 

# OPTIONAL PERMISSIONS

none
